EN
Program, which simulates a primitive banking database, using binary or text files.
Based on database structure: clientData {int accNum; char Sur[15]; char Name[10]; float balance;}
The rules: must have functions, which allow adding or deleting entries, printing all entries, finding an entry, changing the balance of an entry, printing debtors (entries with negative balance), counting the amount of entries in the database, and creating copies of the database.

LV
Programma, kas simulē primitīvu bankas datubāzes darbību izmantojot bināros vai teksta failus.
Balstoties uz datubāzes datu struktūras: clientData {int accNum; char Sur[15]; char Name[10]; float balance;}
Nosacījumi: jābūt funkcijām, kas atļaus pievienot un dzēst ierakstus, izvadīt visus ierakstus, atrast ierakstu, izmainīt konta balansu ierakstam, izvadīt parādniekus (ieraksti ar negatīvu balansu), saskaitīt ierakstus (cik ieraksti ir datubāzē), un izveidot datu bāzes kopiju.
