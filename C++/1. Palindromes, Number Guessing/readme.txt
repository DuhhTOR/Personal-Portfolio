EN
6-digit-palindrome: program, that outputs all 6 digit palindromes.

computer-guesses-number: game, where the user imagines a number between 0 to 100 (including), and the computer will try to guess it in 10 tries.

human-guesses-number: game, where the computer generates a random number between 0 to 100 (including), and the user must guess it by typing it on the keyboard in 10 tries.

palindrome-check: program, that based on the input number 'n', checks, if it is a palindrome.


LV
6-digit-palindrome: programma, kas izvada visus 6 ciparu palindromus.

computer-guesses-number: spēle, kur lietotājs izdomā skaitli intervālā no 0 līdz 100 (ieskaitot), un dators mēģina uzminēt to 10 mēģinājumos.

human-guesses-number: spēle, kur dators ģenerē skaitli intervālā no 0 līdz 100 (ieskaitot), un lietotājam skaitli jāuzmin ievadot minējumu uz klaviatūras 10 mēģinājumos.

palindrome-check: programma, kas pēc dotā ievadītā skaitļa 'n', pārbauda, vai tas ir palindromus.
