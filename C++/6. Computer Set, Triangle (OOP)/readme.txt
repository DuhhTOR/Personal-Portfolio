EN
“Computer Set”: program, which demonstrates class operation and creates a computer set, using class composition of 3 classes - Computer, Monitor, ComputerSet.

“Triangle”: program, which demonstrates class operation and checks, if triangle is possible with given values, calculates the perimeter and area of the triangle, and checks, if the triangle is a right-angled triangle.


LV
“Computer Set”: programma, kas attēlo klašu darbību un veido datora komplektu, veicot kompozīciju no 3 klasēm – Computer, Monitor, ComputerSet.

“Triangle”: programma, kas attēlo klašu darbību un pārbauda, vai trīsstūris ir iespējams ar dotām malām, aprēķina perimetru un laukumu, un pārbauda, vai trīsstūris ir taisnleņķa.
